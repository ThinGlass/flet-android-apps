# Merriam-Webster Dictionary app

Simple application that searches for a word in Merriam-Webster Collegiate Dictionary and returns the definitions.

## API-key setup

1. Create a developer account on the [MW Dictionary API](https://dictionaryapi.com/register/index) page.
2. Create .env file in `src` folder and store the api-key as `COLLEGIATE_DICTIONARY_KEY=...`.
3. Run locally or package the application.
